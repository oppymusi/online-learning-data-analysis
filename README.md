# Online Learning Data Analysis

## Project Overview
This project analyzes user learning behavior on the MasaDepan.ku learning platform. The objective is to identify learning patterns, user engagement, and course completion using data analysis techniques.

## Dataset
- Total records: 5,430
- Source: PT Semesta Integrasi Digital
- Topic: User learning activity on the MasaDepan.ku platform

## Objectives
- Analyze user demographics
- Identify the most popular fields of study
- Analyze course completion rates
- Evaluate learning performance
- Generate business insights and recommendations

## Tools
- Python
- Pandas
- NumPy
- Google Colab
- Tableau

## Data Cleaning
- Data type conversion
- Missing value handling
- Duplicate checking
- Data consistency checking
- Typographical error correction
- Outlier detection

## Exploratory Data Analysis
- User distribution by gender
- User distribution by age
- Popular study fields
- Course ratings
- Final scores
- Learning activity

## Dashboard
The interactive dashboard was developed using Tableau.
